# PHP Quest

Курс по объектно-ориентированному программированию в PHP.

Идём в правильное ООП: маленькие объекты с поведением, композиция вместо наследования где это возможно, декораторы, интерфейсы, стратегии. Всё через короткие практические примеры в знакомой игровой теме.

Этот курс продолжает путь после базового PHP и после захода в ООП через C++. Здесь тот же порядок идей, что и в `cpp-quest`, но без лишнего шума языка: меньше синтаксического трения, больше внимания к самим объектам.

Курс рассчитан на тех, кто уже знаком с основами PHP:
- переменные и типы данных
- строки и массивы
- `if`, `else`, `match`, `foreach`, `for`, `while`
- функции и возврат значений
- работа с файлами на базовом уровне

## Как скачать

```bash
git clone git@github.com:Lemon-s-Home/php-quest.git
cd php-quest
```

## Как запускать примеры

```bash
php file.php
```

Файлы лучше создавать отдельно под каждую задачу. Формат имени:

```text
NN-topic.php
```

Где `NN` — номер урока, а `topic` — 1-2 коротких слова по теме.

Примеры:

```text
01-echo.php
01-variables.php
02-if.php
```

Если нужен конкретный файл урока:

```bash
php 01-echo.php
```

Начиная с урока 14 появляется Composer. Для таких примеров сначала выполни команды из урока (`composer dump-autoload`, `composer require ...`), а потом запускай `main.php`.

## Стиль кода в этом курсе

- пишем `declare(strict_types=1);`
- предпочитаем маленькие функции и маленькие классы
- у объекта должно быть своё поведение, а не просто набор публичных полей
- сначала учимся выражать мысль ясно, потом уже экономить строки
